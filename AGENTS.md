# SYSTEM RULES

Focus ONLY on the requested task.
Do not scan or modify unrelated files.
Do not refactor the entire project unless explicitly requested.

Always:
- Explain the plan first.
- Then generate code.
- Then explain modified files.

Use:
- Modular architecture.
- TypeScript.
- Clean services.
- Async/await.
- Environment variables.

Never:
- Hardcode secrets.
- Create unnecessary abstractions.
- Change working code without reason.
